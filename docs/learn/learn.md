# Learning Splice

Welcome to the **Splice programming language guide**.

Splice is a lightweight scripting language designed for:

- embedded systems
- microcontrollers
- desktop environments
- custom operating systems

It focuses on:

- small runtime
- predictable execution
- fast startup
- easy embedding in C

This guide teaches how to write programs in **Splice (.spl files)**.

## Getting Started

Create a file:

hello.spl

Example program:

    print("Hello, Splice!")

Run it using the runtime:

    ./splice hello.spl
