# Input and Output

Splice provides basic functions for interacting with the user.

## Printing Output

Use `print()` to display text or numbers.

Example:

    print("Hello")
    print(10)
    print(5 + 5)

Example with variables:

    let name = "Developer"

    print("Hello")
    print(name)

## User Input

Splice can read user input.

Example:

    print("Enter your name")

    let name = input()

    print("Hello")
    print(name)
